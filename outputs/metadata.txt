Description: TP2 DevSecOps - Pipeline CI/CD sécurisée pour une application Flask vulnérable.
             Mise en place d'un pipeline GitHub Actions couvrant :
               - SAST (Semgrep)
               - Secrets scanning (Gitleaks)
               - Lint Dockerfile (Hadolint)
               - SCA filesystem (Trivy fs)
               - Image scanning (Trivy image)
             Objectif : détecter et corriger les vulnérabilités (dépendances, Dockerfile, code)
             avant la mise en production.
